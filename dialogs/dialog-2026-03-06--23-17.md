## github copilot with sonnet 4.6:

```
在 PowerShell 7 中依次执行以下命令即可：

```powershell
# 1. 进入 csv-reporter 目录
cd C:\Users\wubin\OOR\katas\dotenv-cpp\csv-reporter

# 2. 配置（已有 build 目录则可跳过，但重新配置无害）
cmake -B build -S .

# 3. 编译
cmake --build build --config Debug

# 4. 运行所有测试（方式一：CTest，推荐）
cd build
ctest -C Debug --output-on-failure
```

或者直接运行测试可执行文件（能看到 GTest 的详细输出）：

```powershell
# 方式二：直接执行测试二进制（在 build 目录下）
.\Debug\csv_reporter_tests.exe
```

**预期输出**（方式二示例）：
```
[==========] Running 30 tests from 10 test suites.
...
[  PASSED  ] 30 tests.
```

方式一的 CTest 会给出：
```
100% tests passed, 0 tests failed out of 30
```

> 如果 build 目录已存在（上次已构建），直接从第 3 步开始即可。
```
